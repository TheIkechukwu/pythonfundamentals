🐍 Python Basics
Python is a readable, beginner-friendly language. Here are the building blocks every learner starts with.
💻Your First Python Program

The classic starting point — printing text to the screen.

# This is a comment — Python ignores it
print("Hello, World!")
print("Welcome to Learn With iKay!")
Beginner Output: Hello, World!
📌Variables & Assignment

Variables store values. No need to declare types!

name = "Ada"           # string
age = 25               # integer
height = 1.65          # float
is_student = True     # boolean

print(name, age, height, is_student)
# Output: Ada 25 1.65 True
Tip: Variable names are case-sensitive
⌨️Getting User Input
name = input("Enter your name: ")
print("Hello,", name)

age = int(input("Enter your age: "))  # convert to int
print("You are", age, "years old")
Note: input() always returns a string
🖨️Print Formatting
name = "Chidi"
score = 92.5

# f-string (modern, recommended)
print(f"Student: {name}, Score: {score}")

# .format() method
print("Student: {}, Score: {}".format(name, score))

# % formatting (older style)
print("Student: %s, Score: %.1f" % (name, score))

🔍Comparison & Logical
x, y = 10, 5

# Comparison (returns True/False)
print(x == y)   # False — equal to
print(x != y)   # True  — not equal
print(x > y)    # True  — greater than
print(x <= y)   # False — less or equal

# Logical
print(x > 5 and y < 10)  # True
print(x < 5 or  y < 10)  # True
print(not (x == y))        # True
🔀 Control Flow
Control flow lets your program make decisions and repeat actions.
❓if / elif / else
score = 75

if score >= 90:
    print("Grade: A")
elif score >= 70:
    print("Grade: B")    # ← this runs
elif score >= 50:
    print("Grade: C")
else:
    print("Grade: F")
Indentation (4 spaces) is mandatory in Python!
🔁for Loop
# Loop over a range
for i in range(1, 6):
    print(i)          # prints 1 2 3 4 5

# Loop over a list
fruits = ["apple", "mango", "guava"]
for fruit in fruits:
    print(f"I like {fruit}")

# enumerate — get index AND value
for idx, fruit in enumerate(fruits):
    print(f"{idx}: {fruit}")
🔂while Loop
count = 0
while count < 5:
    print(f"Count: {count}")
    count += 1

# break — exit loop early
for n in range(10):
    if n == 5:
        break
    print(n)

# continue — skip to next iteration
for n in range(6):
    if n == 3:
        continue
    print(n)  # skips 3
🔧 Functions
Functions are reusable blocks of code. They keep your programs clean and DRY (Don't Repeat Yourself).
📐Defining & Calling Functions
def greet(name):
    """Says hello to someone."""  # docstring
    print(f"Hello, {name}!")

greet("Adesuwa")   # → Hello, Adesuwa!

# Function with return value
def add(a, b):
    return a + b

result = add(3, 7)
print(result)      # → 10
🎛️Default & Keyword Arguments
def introduce(name, role="student"):
    print(f"{name} is a {role}")

introduce("Emeka")                  # Emeka is a student
introduce("Mr Kola", "teacher")     # Mr Kola is a teacher
introduce(role="coach", name="Yemi") # keyword order flexible

# *args — any number of positional args
def total(*nums):
    return sum(nums)

print(total(1, 2, 3, 4))  # → 10
⚡Lambda Functions
# Anonymous one-liner functions
square = lambda x: x ** 2
print(square(5))  # → 25

add = lambda a, b: a + b
print(add(3, 4))   # → 7

# Common use: sorting
students = [("Temi", 85), ("Bisi", 92), ("Kola", 78)]
students.sort(key=lambda s: s[1], reverse=True)
print(students)  # sorted by score, highest first
📋 Lists, Tuples, Sets & Dicts
Python's collection types let you store and organize groups of data efficiently.
📝Lists — Ordered & Mutable
fruits = ["apple", "banana", "mango"]

# Access — zero-indexed
print(fruits[0])    # apple
print(fruits[-1])   # mango (last)

# Modify
fruits.append("guava")   # add to end
fruits.insert(1, "pear") # insert at index
fruits.remove("banana") # remove by value
popped = fruits.pop()   # remove & return last

# Slicing
print(fruits[1:3])   # items at index 1 and 2
print(fruits[::-1]) # reversed list

# Useful methods
print(len(fruits))          # length
print(sorted(fruits))       # sorted copy
print("apple" in fruits)   # membership check
🗂️Dictionaries — Key : Value
student = {
    "name": "Ngozi",
    "age": 22,
    "grade": "A"
}

# Access
print(student["name"])          # Ngozi
print(student.get("age"))      # 22 (safe, no error)

# Add / Update
student["email"] = "ngozi@email.com"
student["age"] = 23

# Loop
for key, val in student.items():
    print(f"{key}: {val}")

# Keys, values
print(student.keys())
print(student.values())
📦Tuples & Sets
# Tuple — immutable (cannot change)
coords = (6.4550, 3.3841)  # latitude, longitude
lat, lon = coords          # unpacking
print(f"Lat: {lat}, Lon: {lon}")

# Set — unique values only
colors = {"red", "green", "blue", "red"}
print(colors)    # {'red', 'green', 'blue'} — no duplicate

colors.add("yellow")
colors.discard("red")

# Set operations
a = {1,2,3}; b = {2,3,4}
print(a | b)   # union: {1,2,3,4}
print(a & b)   # intersection: {2,3}
print(a - b)   # difference: {1}

✂️Slicing & Indexing
word = "PROGRAMMING"
#       0123456789...

print(word[0])        # P  — first character
print(word[-1])       # G  — last character
print(word[0:4])      # PROG

print(word[::2])      # PORMIG — every 2nd char

print(len(word))      # 11
📁 Files & Error Handling
Reading/writing files and handling errors gracefully are essential real-world skills.
📂Reading & Writing Files
# Write to a file
with open("notes.txt", "w") as f:
    f.write("Hello from Python!\n")
    f.write("Learn With iKay\n")

# Read entire file
with open("notes.txt", "r") as f:
    content = f.read()
    print(content)

# Read line by line
with open("notes.txt", "r") as f:
    for line in f:
        print(line.strip())

# Append to file
with open("notes.txt", "a") as f:
    f.write("New line added!\n")
ModeMeaning
"r"Read (default)
"w"Write (overwrites)
"a"Append
"rb"Read binary
🛡️Try / Except — Error Handling
try:
    num = int(input("Enter a number: "))
    result = 100 / num
    print(f"Result: {result}")

except ValueError:
    print("That's not a valid number!")

except ZeroDivisionError:
    print("Cannot divide by zero!")

except Exception as e:
    print(f"Unexpected error: {e}")

else:
    print("No errors! Great.")     # runs if no exception

finally:
    print("This always runs!")     # always executes
